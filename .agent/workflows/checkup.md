# Project Checkup Workflow

1. Scan `src/backend` and `src/frontend` for consistency.
2. Check database migrations in `src/migrations`.
3. Verify `package.json` and `requirements.txt` dependencies.
4. Provide a summary of implemented features and missing parts.
